# Personal-Expense-Tracker
This is a basic "Personal EXPENSE Tracker" built in Python to store details of one's expenses.

This is built using a list of dictionaries to track the expenditures.
The "sleep" function of the "time" module is also used.

The user can enter his/her choice and perform the following operations : 

1. Adding a new expense 
2. View all the expenses so far 
3. View the total amount spent till now 
4. Exit the current track.
